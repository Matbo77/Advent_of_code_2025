# Advent_of_code_2025
My solutions to the Advent of Code 2025 written in Python and C++.

From day 1 to day 6 - (Work in progress)

Each folder contains:

- The solution script
- Test puzzle input
- Real puzzle input


To execute the C++ file:
- Open a terminal in your .cpp file folder
- ``` g++ -g -o advent_of_code_dayX dayX.cpp ```
- ``` ./advent_of_code_dayX ```
